
## Template Usage Guide

### How to Use These Templates

1. **Personalization**
   - Replace all `{{variables}}` with actual campaign and company data
   - Use actual contact names and specific metrics (avoid generic language)
   - Include relevant dates and specific numbers

2. **Tone Adjustment**
   - Capacity Expansion: Urgent but supportive
   - High Performance: Celebratory and confident
   - Bundle Consolidation: Pragmatic and solution-focused
   - Engagement Boost: Collaborative and advisory
   - Strategic Planning: Consultative and visionary

3. **Segmentation Strategy**

   Use this template selection logic:

   ```
   IF campaign.capacityUtilization >= 0.80 AND campaign.status != 'completed'
      → Use "Capacity Expansion Opportunity"

   IF campaign.cumulativeSROI > 5 OR campaign.averageVIS > 80
      → Use "High Performance Recognition"

   IF company.activeCampaigns >= 2 AND portfolio.highPerformingCount >= 1
      → Use "Bundle Consolidation Proposal"

   IF campaign.currentSessions > 100 AND campaign.cumulativeSROI < 3
      → Use "Engagement Boost Opportunity"

   IF company.activeCampaigns >= 3
      → Use "Multi-Campaign Growth Strategy"

   IF quarter_end
      → Use "End-of-Quarter Expansion Planning"
   ```

4. **Timing Recommendations**

   | Template | Best Day | Best Time | Frequency |
   |----------|----------|-----------|-----------|
   | Capacity Expansion | Tuesday-Thursday | Morning | Trigger-based |
   | High Performance | Monday or Wednesday | Mid-morning | Monthly |
   | Bundle Consolidation | Tuesday | Afternoon | Quarterly |
   | Engagement Boost | Wednesday-Thursday | Morning | Trigger-based |
   | Multi-Campaign Growth | Monday-Tuesday | Morning | Semi-annual |
   | Quarterly Planning | Last week of quarter | Any | Quarterly |

5. **Success Metrics**

   Track these metrics for each email campaign:
   - Open rate (target: >35%)
   - Click rate (target: >15%)
   - Reply rate (target: >25%)
   - Meeting scheduled: >50% of replies
   - Deal conversion within 90 days: >30%
