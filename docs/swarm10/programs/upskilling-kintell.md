## Product Overview

**Upskilling (Kintell)** is a career development platform providing technical skills courses, certifications, and job placement support for underrepresented talent (refugees, women, low-income).

**Tagline**: *"From Underemployed to Upskilled – Career Transformation"*

---

## The Problem

**48% of underrepresented talent are underemployed** (skills below job level):
- Refugees with degrees work in low-skill jobs (e.g., engineer → warehouse)
- Women face skills gaps in tech (70% lack coding skills)
- Low-income workers stuck in dead-end roles (no upskilling access)

**Result**: $25K avg salary gap (underemployed vs. skilled roles)

---

## The Solution

**Upskilling (Kintell)** provides:
- **Tech Skills**: Coding (Python, JavaScript), data analysis, cloud (AWS, Azure)
- **Soft Skills**: Leadership, communication, project management
- **Certifications**: AWS Certified, Google Analytics, PMI, etc.
- **Job Placement**: Resume support, interview prep, employer matching

**Outcome**: 72% achieve certification; 58% promoted or find new job within 9 months

---

## How It Works

### 1. Skills Assessment
- **Current Skills**: Tech stack, certifications, experience level
- **Career Goal**: Target role (e.g., "Data Analyst")
- **Learning Path**: Personalized curriculum (courses + certs)

**Example**:
```
Learner: Laila (refugee, BA Economics, goal: Data Analyst)
Path: SQL → Python → Google Analytics Cert → Excel Advanced
Duration: 9 months, 8 hours/week
```

### 2. Course Catalog
- **Tech Skills**: 120+ courses (Python, SQL, AWS, JavaScript, etc.)
- **Soft Skills**: Leadership, communication, negotiation
- **Certifications**: AWS, Google, Microsoft, PMI

**Platform Features**:
- Video lessons + hands-on labs
- Progress tracking + badges
- Peer community (learner cohorts)
- Tutor support (1:1 sessions)

### 3. Impact Measurement
- **Certification Rate**: 72% achieve target cert
- **Promotion Rate**: 38% promoted within company
- **New Job Rate**: 20% find new job (higher salary)
- **Salary Lift**: $18K avg increase
- **SROI**: 3.8

**SROI Calculation**:
```
Economic Value: $18K salary lift × 58% success × 3 years = $31,320
Program Cost: $8,250 (courses $6K + tutor $2K + platform $250)
SROI: $31,320 / $8,250 = 3.80
```
