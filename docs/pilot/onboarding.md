## Getting Started

### First Login

When you first log in to the Corporate Cockpit, you'll be automatically directed to the welcome page at:

- **English**: `/en/cockpit/{companyId}/welcome`
- **UK English**: `/uk/cockpit/{companyId}/welcome`
- **Norwegian**: `/no/cockpit/{companyId}/welcome`

The welcome page provides a guided tour of the platform and helps you complete essential setup tasks.

### User Roles

The platform supports several user roles with different permissions:

- **Company Admin**: Full access to all features including settings, user management, and theme customization
- **Company User**: Access to create and edit programs, view reports, and export data
- **Viewer**: Read-only access to dashboards and reports
- **System Admin**: Platform-wide administrative access (internal use only)

## Welcome Flow

The welcome flow is a 4-step guided tour that appears on your first login:

### Step 1: Welcome

Introduces the platform and its core value propositions:
- **Track Your Impact**: Measure SROI for CSR programs
- **Engage Your Volunteers**: Monitor VIS scores and recognize contributors
- **Evidence-Based Reporting**: Transform qualitative feedback into quantitative evidence (Q2Q)

### Step 2: Choose Your Theme

Select a pre-configured color scheme that matches your brand:

- **Corporate Blue**: Professional blue/gray palette for technology companies
- **Healthcare Green**: Calming green/white palette for healthcare and sustainability
- **Finance Gold**: Sophisticated gold/navy palette for financial services
- **Modern Neutral**: Versatile grayscale palette suitable for any industry
- **Community Purple**: Warm purple palette for non-profits and social impact organizations

All themes meet **WCAG 2.2 AA accessibility standards**.

> **Note**: You need admin permissions to save theme changes permanently.

### Step 3: Key Features Tour

Overview of the main features available in the Corporate Cockpit:

- **Dashboard Widgets**: Real-time KPI cards with SROI, VIS scores, and participation metrics
- **Report Generation**: Professional PDF and PowerPoint exports with watermarks and evidence IDs
- **Evidence Tracking**: View Q2Q evidence with lineage trails and confidence scores
- **Team Management**: Invite team members and manage role-based access control

### Step 4: Getting Started Checklist

Preview of the onboarding tasks you'll want to complete to fully set up your workspace.

### Skipping the Welcome Flow

You can skip the welcome flow at any time by clicking "Skip tour". If you check "Don't show this again", the welcome flow won't appear on future logins. You can always access the checklist from your dashboard.

## Theme Customization

### Accessing Theme Settings

Company admins can access theme customization in two ways:

1. **During welcome flow**: Step 2 allows you to preview and select a theme
2. **Admin settings**: Navigate to `/en/cockpit/{companyId}/admin` and go to the Branding section

### Theme Presets

Each preset includes:

- **Color palette**: Primary, secondary, and accent colors
- **Typography**: Font family and weights
- **Spacing**: Scale multiplier for consistent spacing
- **WCAG compliance**: All presets meet AA contrast requirements

### Custom Themes

Beyond presets, company admins can:

- Upload company logo
- Fine-tune individual colors
- Adjust typography settings
- Configure dark mode preferences (coming soon)

## Getting Started Checklist

The checklist helps you complete essential onboarding tasks:

### Task 1: Upload Company Logo
**What**: Add your company logo to personalize reports and dashboards
**Where**: Admin Settings > Branding
**Impact**: Makes reports more professional and brand-consistent

### Task 2: Configure SSO
**What**: Set up single sign-on for your organization
**Where**: Admin Settings > SSO
**Impact**: Streamlines user access and improves security

### Task 3: Invite Team Members
**What**: Add colleagues and assign appropriate roles
**Where**: Admin Settings > Users
**Impact**: Enables team collaboration on CSR initiatives

### Task 4: Create First Program
**What**: Set up a buddy, kintell, or upskilling program
**Where**: Programs > New Program
**Impact**: Begins tracking impact data for your initiatives

### Task 5: View First Report
**What**: Explore SROI and VIS dashboards
**Where**: Reports section
**Impact**: Understand how metrics are calculated and visualized

### Task 6: Export First PDF
**What**: Generate a professional impact report
**Where**: Reports > Export
**Impact**: Learn how to create executive-ready reports

### Checklist Features

- **Progress tracking**: Automatically saved in your browser
- **Collapsible**: Minimize to reduce clutter when not needed
- **Dismissable**: Hide permanently once completed
- **Restorable**: Bring back dismissed checklists if needed

### Common Questions

**Q: Can I change my theme after selecting one?**
A: Yes! Company admins can change themes anytime via Admin Settings > Branding.

**Q: What happens to the checklist after I complete it?**
A: It stays on your dashboard until you dismiss it. You can restore it anytime.

**Q: Can I skip the welcome flow and come back later?**
A: Yes, but the flow is designed to help you get started quickly. If skipped, focus on the checklist instead.

**Q: How do I add more users?**
A: Go to Admin Settings > Users and click "Invite Team Member". They'll receive an email invitation.

**Q: Are theme changes visible to all users?**
A: Yes, theme changes apply to all users within your company tenant.

**Q: Can viewers export reports?**
A: No, only Company Users and Admins have export permissions.
